I primarily work on engineering practices around NLP and large language models, with a focus on usable, deployable, and scalable application solutions:

- Agent framework design and multi-tool orchestration (LangGraph / MCP / Human-in-the-Loop)
- RAG knowledge base construction (hybrid retrieval: vector + keyword + graph)

## Featured Projects

- [DeepClaw](https://github.com/shell-nlp/deepclaw)
  Open-source Agent / RAG full-stack scaffold that integrates a general-purpose Agent, RAG knowledge base, skill management, multi-channel integration, MCP configuration, and frontend UI into a single FastAPI service. Supports LangGraph Agent orchestration, Elasticsearch hybrid retrieval, OpenSandbox multi-user container isolation, Human-in-the-Loop approval workflows, AG-UI protocol, and Feishu/DingTalk/WeChat ClawBot channel routing.

- [gpt_server](https://github.com/shell-nlp/gpt_server)
  LLM inference service providing OpenAI-compatible APIs, supporting unified local and server-side inference with built-in model hot-loading and multi-route concurrent scheduling.

- [openai_router](https://github.com/shell-nlp/openai_router)
  Unified routing gateway for multi-model, multi-provider scenarios, managing different model sources through a single OpenAI-compatible interface with load balancing and failover support.

- [LLMs_finetuning](https://github.com/shell-nlp/LLMs_finetuning)
  Open-source LLM fine-tuning practices covering LoRA / QLoRA / PEFT adaptation methods and complete training engineering pipelines.
